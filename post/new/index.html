NiceGUI

2024-12-16 2024-12-16

NiceGUI NiceGUI 是一个易于使用、基于 Python 的 UI 框架，它显示在您的 Web 浏览器中。您可以创建按钮、对话框、Markdown、3D 场景、绘图等等。 它非常适合微型 Web 应用程序、仪表板、机器人项目、智能家居解决方案和类似用例。您还可以在开发中使用它，例如在调整/配置机器学习算法或调整电机控制器时。 NiceGUI 的 PyPI package, Docker image 和 conda-forge以及 GitHub 特征 1.基于浏览器的图形用户界面 2.代码更改时隐式重新加载 3.充当 Web 服务器（由浏览器访问）或处于本机模式（例如桌面窗口） 4.标准 GUI 元素，如标签、按钮、复选框、开关、滑块、输入、文件上传…… 5.使用行、列、卡片和对话框进行简单分组 6.通用 HTML 和 Markdown 元素 7.强大的高级元素 1.绘制图形和图表， 2.渲染 3D 场景， 3.通过虚拟操纵杆获取转向事件 4.注释和叠加图像 5.与表交互 6.导航可折叠树结构 7.嵌入视频和音频文件 8.内置计时器，可每隔一段时间（甚至每 10 毫秒）刷新数据 9.直接的数据绑定和可刷新函数，可编写更少的代码 1.通知、对话框和菜单，提供最先进的用户交互 2.共享网页和个人网页 1.易于使用的每用户和一般持久性 2.能够添加自定义路由和数据响应 3.捕获全局快捷键的键盘输入等。 4.通过定义原色、辅助色和强调色来自定义外观 5.Live-Cycle 事件和会话数据 6.在 Jupyter Notebook 中运行，并允许 Python 的交互模式 7.对 Tailwind CSS 的自动完成支持 8.SVG、Base64 和表情符号网站图标支持 9.基于 pytest 的测试框架 安装 python3 -m pip install nicegui 用法 在文件 main.py 中编写漂亮的 GUI： 从 NiceGUI 导入 UI ui.label（‘Hello NiceGUI！’） ui.button（‘BUTTON’， on_click=lambda： ui.notify（‘按钮被按下’）） ui.run（） 使用以下方法启动它： python3 main.py GUI 现在可通过浏览器中的 http://localhost:8080/ 使用。注意：当您修改代码时，NiceGUI 将自动重新加载页面。 文档和示例 该文档托管在 https://nicegui.io/documentation 上，并提供了大量实时演示。https://nicegui.io 的全部内容 都是通过 NiceGUI 本身实现的，可以使用 docker run -p 8080：8080 zauberzeug/nicegui 在本地启动，也可以通过执行此存储库中的 main.py 来启动。 您还可以查看我们的深入示例，了解您可以使用 NiceGUI 做什么。在我们的 wiki 中，我们列出了来自社区的优秀 NiceGUI 项目，一个包含教程的部分，一个不断增长的常见问题解答列表，以及一些使用 ChatGPT / LLM 获取有关 NiceGUI 的帮助的策略。 为什么？ 我们 Zauberzeug 喜欢 Streamlit，但发现它在状态处理方面发挥了太多的魔力。在寻找替代的优秀库来用 Python 编写简单的图形用户界面时，我们发现了 JustPy。尽管我们喜欢这种方法，但它对于我们的日常使用来说太“低级 HTML”了。但它激发了我们使用 Vue 和 Quasar 作为前端。 我们建立在 FastAPI 之上，FastAPI 本身基于 ASGI 框架 Starlette 和 ASGI Web 服务器 Uvicorn，因为它们具有出色的性能和易用性。 赞助商 所有贡献者和赞助商都可以维护该项目。如果您想支持此项目并让您的头像或公司徽标出现在下方，请赞助我们。 💖
